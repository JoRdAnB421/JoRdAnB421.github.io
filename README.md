# Your GitHub Pages Site

## File structure
```
site/
├── index.html        ← Homepage (start here)
├── style.css         ← All shared colours, fonts, nav, footer
├── cv/
│   └── index.html    ← CV page
├── games/
│   └── index.html    ← Games / itch.io projects
└── outreach/
    └── index.html    ← Talks + YouTube channel
```

## How to go live
1. Create a GitHub account if you don't have one
2. Create a new repo named exactly: `yourusername.github.io`
3. Upload all these files (keeping the folder structure)
4. Wait ~60 seconds — your site is live at https://yourusername.github.io

## What to edit first (search for "↓ Edit" in any file)
- Your name: change "YourName" everywhere
- Your email: change "you@email.com"
- Your handles: change "yourhandle" in all the social links
- index.html: edit the about bio paragraphs
- cv/index.html: fill in your jobs, education, skills, publications
- games/index.html: replace the placeholder games with your real itch.io projects
- outreach/index.html: list your actual talks
